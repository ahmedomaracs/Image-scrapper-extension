# Image Scraper Extension

##  Overview
A Chrome extension that extracts all images from any webpage and displays them in a clean popup.

##  Features
- Scrape all images instantly
- Clean popup UI
- Fast and lightweight

##  Tech Stack
- JavaScript
- HTML
- CSS

## Installation
1. Download the repo
2. Open Chrome → Extensions
3. Enable Developer Mode
4. Click "Load unpacked"
5. Select the project folder
